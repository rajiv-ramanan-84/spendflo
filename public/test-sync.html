Curl Command Examples

1. Upload File
curl -X POST http://localhost:3000/api/sync/upload \ -F "file=@/path/to/your/file.xlsx" \ -F "customerId=test_customer_123"
2. Trigger Sync
curl -X POST http://localhost:3000/api/sync/direct \ -H "Content-Type: application/json" \ -d '{"customerId": "test_customer_123"}'
3. Check Files in Upload Directory
curl http://localhost:3000/api/sync/debug?customerId=test_customer_123
